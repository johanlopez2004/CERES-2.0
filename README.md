# CERES 2.0

CERES 2.0 es una versión actualizada de la aplicación móvil CERES diseñada para apoyar la gestión de huertas urbanas comunitarias.

La aplicación permite a los usuarios registrarse, crear y administrar huertas, colaborar con otros miembros de la comunidad y acceder a información compartida. El sistema utiliza Firebase como backend para la autenticación de usuarios, almacenamiento de datos y gestión de archivos.

## Tecnologías utilizadas
- Android (Java)
- Firebase Authentication
- Firebase Firestore
- Firebase Storage
- Android Studio

## Funcionalidades principales
- Registro e inicio de sesión de usuarios
- Creación y gestión de huertas
- Colaboración entre usuarios
- Almacenamiento de información en la nube
- Sincronización de datos con Firebase
